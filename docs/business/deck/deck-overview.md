# Deck Overview

## Vai trò
Deck là đơn vị chứa flashcard và là đơn vị học cụ thể nhất.

## Đặc điểm
- Một deck chứa nhiều flashcard
- User có thể học trực tiếp từ deck
- Deck không phải đơn vị tổ chức cây
- Hỗ trợ tạo, sửa, xóa, di chuyển, duplicate deck
- Hỗ trợ import flashcard vào deck
- Hỗ trợ export toàn bộ deck
- Hỗ trợ tìm kiếm và sắp xếp danh sách deck

## Khi học từ deck
- Chỉ lấy flashcard trong deck đó
